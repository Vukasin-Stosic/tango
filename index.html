The Binary Puzzle

The binary puzzle is a challenging logic puzzle that can be solved just by reasoning. Only zeros and ones occur in the puzzle, but this turns out to be more complicated than it seems to be. A last warning: The binary puzzle is quite addictive.

Rules

Each cell should contain a zero or a one. No more than two similar numbers below or next to each other are allowed. Each row and each column is unique and contains as many zeros as ones.
